# RoomAI Frontend

React + Vite UI built with Tailwind CSS v3 and shadcn/ui components, including a drag-and-drop layout editor.

## Local Dev

```bash
npm install
npm run dev
```

## Environment

Create `frontend/.env` and set:

```
VITE_API_URL=http://localhost:8000
```

If no API is set, the UI uses mock data.
